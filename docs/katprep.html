katprep.get_credentials(prefix, hostname=None, auth_container=None, auth_pass=None)[source]¶

Retrieves credentials for a particular external system (e.g. Satellite). This function checks whether a hostname is part of an authentication container or retrieves credentials from an authentication file. If both approaches fail, logon credentials are prompted.

Parameters:
  • prefix (str) – prefix for the external system (used in variables/prompts)
  • hostname (str) – external system hostname
  • auth_container (str) – authentication container file name
  • auth_pass (str) – authentication container password
